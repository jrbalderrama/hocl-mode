# hocl-mode

HOCL mode for Emacs.

![alt Screenshot](https://github.com/jrbalderrama/hocl-mode/blob/master/screenshot.png)

## Features
 - Highlight support
 - Smart comment using ```comment-dwim```

## Install

Copy the ```hocl-mode.el``` file where Emacs (```'load-path```) can load it.
